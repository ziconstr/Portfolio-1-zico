## Getting Started

### Installing

Get started with this project.

1. Clone the repo:

```bash
git clone git@github.com:ziconstr/Portfolio-1-zico.git
```

2. Install the dependencies:

```
npm install
```

### Running

To run the app, run the following commands:

```bash
npm run start
```
